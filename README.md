# FDNAssessment

> Simple Web for Developer Skill Assessment

## Quick Start

``` bash
# Install dependencies
npm install express body-parser sequelize pg pg-hstore express-handlebars
npm install -D handlebars@4.5.0

# Serve on localhost:3000
npm start
# Dev Server (Nodemon)
npm run dev
```
